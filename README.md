# chrome_password

~~Steal~~ Get username & password from Chrome. (**Now Only Windows**)

## Try it

```shell
npx chrome_password
```

## Installation

```powershell
npm install -g chrome_password
```

or

```powershell
yarn global add chrome_password
```
